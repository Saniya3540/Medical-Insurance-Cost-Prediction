# Medical Insurance Cost Prediction

Machine Learning project that predicts medical insurance charges using Linear Regression.

## Features
- Data preprocessing
- Correlation heatmap
- Model training using Linear Regression
- Prediction and evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- Linear Regression

## Evaluation Metrics
- R2 Score: 0.78
